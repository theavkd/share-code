F12 Overrides: Chưa kết nối

Danh Sách File Mồi

Bắt Cả Đàn Cá 🐟 Fetch và gộp tất cả file cùng lúc

Kei hướng dẫn cậu nè:

  1. Nhấn F12 $\rightarrow$ tab Sources $\rightarrow$ Overrides.
  2. Chọn thư mục chứa dự án dưới máy cậu (ví dụ thư mục contact). Nhớ bấm Allow (Cho phép) nhé.
  3. Tạo file tương ứng (vd: main.py) ngay trong thư mục đó rồi lưu lại ở VS Code.
  4. Quay lại đây nhấn F5 hoặc bấm nút Tải lại file để nạp code nha!
